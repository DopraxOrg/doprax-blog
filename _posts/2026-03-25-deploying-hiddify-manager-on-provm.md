### ✅ Step 1: Launch a ProVM (30 seconds)

1. Log in to your [Doprax Dashboard](https://app.doprax.com)
2. Click **+ New Resource** → **ProVM**
3. Choose:
   - Provider: `Hetzner` (recommended for speed + value) or `Vultr` / `Gcore` (for global proximity)
   - Region: `FRA`, `HEL`, `LGA`, or `SFO` (pick closest to your users)
   - Plan: `CX21` (2 vCPU, 4 GB RAM, 80 GB SSD) — enough for 500+ concurrent users
   - OS: `Ubuntu 24.04 LTS` (fully supported by Hiddify)
   - Enable **Auto-Backup** (optional but recommended)
4. Click **Deploy**

Your VM boots in <12 seconds. No waiting for IP assignment. No manual SSH key setup — Doprax injects your keys automatically.

> 💡 Pro tip: All ProVMs come with IPv4 + IPv6, full root access, and unrestricted outbound/inbound ports — no carrier-grade NAT, no port blocking. This matters for Hiddify’s TLS fallback and UDP-based protocols.

---

### ✅ Step 2: Run the Official Hiddify Installer (60 seconds)

Once the VM status shows **Running**, click its name → go to the **Terminal** tab (web-based, no local SSH client needed).

Paste and run this single command:

```bash
curl -fsSL https://hiddify.com/install.sh | bash -s -- -d
```

That’s it. The script:
- Installs Docker, Docker Compose, and required system packages
- Pulls the official `hiddify-manager` image (`ghcr.io/hiddify/hiddify-manager:latest`)
- Generates TLS certificates via Let’s Encrypt (or uses your own domain if provided)
- Starts the service behind an embedded Nginx reverse proxy with automatic HTTP→HTTPS redirect

No editing config files. No `systemctl enable`. No certbot manual mode.

> 🛑 Don’t use `--no-tls` unless you’re testing locally. Doprax ProVMs support Let’s Encrypt out of the box — no port 80/443 forwarding hacks required.

---

### ✅ Step 3: Access & Configure (2 minutes)

After ~45 seconds, the terminal outputs:

```
✅ Hiddify Manager is ready!
🌐 Admin Panel: https://<your-vm-ip>
🔑 Default login: admin / <auto-generated-password>
📝 Save this password — it won’t be shown again.
```

Go to that URL in your browser. You’ll see a clean, dark-mode admin dashboard — fully responsive, zero tracking, no telemetry.

From here:
- Add your domain (e.g., `proxy.yourdomain.com`) under **Settings → Domain**
- Enable **Auto Renew TLS** (enabled by default)
- Import configs via QR code, subscription link, or manual JSON
- Enable **WireGuard mode**, **Xray Reality**, or **Trojan-Go** with one toggle
- Set up user quotas, expiry dates, and per-user download limits

All state is persisted in `/opt/hiddify-manager/config` — backed up automatically if you enabled ProVM backups.
